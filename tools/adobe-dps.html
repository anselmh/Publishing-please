Adobe DPS

Description

The Adobe Digital Publishing Suite is the leading solution for digital magazine creations. It is made by Adobe Systems and is updated every single months with new features and bugfixes. Many publishers already rely on this software and WoodWing and vjoon K4 use this technology for their publishing-system.

You can find more information on: adobe.com

Feature list

  • iPad, Android, Blackberry, Kindle Fire, iPhone
  • Apple Newsstand
  • InApp-Purchase (own Kiosk)
  • InDesign Multi-State-Objects
  • InDesign native panels
  • extensions via HTML5
  • updateable issues
  • nested interactivity possible
  • easy layouting with InDesign
  • Multi-Renditions (diff. folios for diff. resolutions)
  • preview on device (no PhoneGap req., available since v20)

Disadvantages

  • not extendable via XCode
  • only online-solution
  • no auto-layout (orientation)
  • no free distribution
  • working with InDesign for interactive content isn’t that easy some time

software requirements

  • Adobe InDesign CS5 / CS5.5 / CS6
  • Adobe Digital Publishing Suite Plugins

Pricing

  • Single Edition: $395 USD / folio (unlimited Single Apps included in Creative Cloud membership)
  • Professional Edition: $495 USD / month
  • Enterprise Edition $2.875 USD / month
